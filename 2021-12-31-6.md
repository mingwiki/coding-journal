# Filter Functions

- brightness()

```css
.avatar:hover {
  filter: brightness(150%);
}
```

- blur()

```css
.ghost {
  filter: blur(50%);
}
```

- contrast()

```css
.wow {
  filter: contrast(200%);
}
```

- grayscale()

```css
.no-color {
  filter: grayscale(100%);
}
```

- invert()

```css
@media (inverted-colors: inverted) {
  img,
  video {
    filter: invert(100%);
  }
}
```

- opacity()

```css
.filter-visibility {
  filter: opacity(0);
}
```

- saturate()

```css
.full-color {
  filter: saturate(100%);
}
```

- sepia()

```css
.is-old-timey {
  filter: sepia(1);
}
```

- drop-shadow()

```css
.fit-shape-shadow {
  filter: drop-shadow(3rem 0 0.5rem #e486da);
}
```

- hue-rotate()

```css
.change-color {
  filter: hue-rotate(180deg);
}
```
